## Software
### Començar a editar

Podeu baixar el projecte complert i descomprimir-lo. Per editar-lo hi ha diverses opcions, jo he estat fent servir: 
* [MikTex](https://miktex.org/help) - Paquets. Conté multitud de llibreries que podem cridar a TexMaker.
* [TeXMaker](https://www.xm1math.net/texmaker/download.html) - Editor de codi. Alhora permet visualitzar el PDF resultant.

### Recomenacions

* [Exemples i plantilles](https://www.latextemplates.com/) - Aquí podeu veure alguns dels documents que es poden fer en TeX, així com els seus codis font.
* [Tutorial](https://www.youtube.com/watch?v=gFAQd0ueIMc) - Tutorial amb el qual vaig començar a programar la plantilla.
* [Autohotkey](https://www.autohotkey.com/) - Programa per configurar macros. Pot ser molt útil per teclejar caràcters o estructures que es facin servir habitualment. 

Per exemple, el backslash (\) es pot treure per pantalla amb ALT+092 o bé amb un petit script fent CTRL+D:
```
^d::
Send, \
return
```
